# Словарь компании Тензор

Navigation: [[index]] | [[Tensor-Company]]

Корпоративная терминология Тензор/Saby для новых сотрудников.

## Продукт

| Термин | Определение |
|---|---|
| **Saby (СБИС) / онлайн** | Основной рабочий ресурс сотрудников — [online.saby.ru](https://online.saby.ru/) |
| **Аккаунт** | Кабинет организации в Saby; место хранения данных; доступен только зарегистрированным пользователям |
| **Аккордеон** | Меню с основными разделами Saby |
| **Демка / демоверсия** | Пробная версия Saby с возможностью просмотра возможностей разных лицензий |

## Бизнес-процессы

| Термин | Определение |
|---|---|
| **БП (бизнес-процесс)** | Последовательность действий в системе для решения бизнес-задач |
| **Веха** | Контрольная точка с ожидаемым результатом (утверждение, завершение, подпись) |
| **Видеосовещание** | Групповой видеозвонок с назначенным временем, повесткой и итогами |
