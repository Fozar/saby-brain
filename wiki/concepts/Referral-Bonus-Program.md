# Referral Bonus Program (Реферальная бонусная программа)

> [!note] Не путать с [[ReferralDeals-System]]
> Эта страница — о **бонусной** реферальной программе (покупатели рекомендуют заведение друзьям и получают бонусы). [[ReferralDeals-System]] — о B2B реферальной системе сделок (заявки в банки на РКО через SabyNet).

---

## Концепт

Реферальная бонусная программа — специализированный тип бонусной программы (`ВидЦены.Тип = 9`).

- Является разновидностью бонусной программы с особыми правилами начисления.
- Может действовать только в точках продаж с подключённой бонусной программой.
- Максимально 3 уровня иерархии (глубина сети рефералов).
- Владелец заведения настраивает: кол-во бонусов по уровням, правила сгорания реферальных баллов.

---

## Информационная модель

| Поле | Значение |
|------|---------|
| Сущность | `ВидЦены` |
| Тип | `9` |
| Начисление | На персональную карту покупателя |
| Расходование | По общим правилам бонусных программ |

---

## Правила начисления

### 1. Бонусы за рекомендацию (от реферала)

| Событие | Момент начисления |
|---------|-----------------|
| Просмотр реферальной ссылки | После перехода приглашённого по ссылке |
| Регистрация в программе | После регистрации приглашённого |
| Первая покупка | После первой покупки приглашённого |

### 2. Бонусы за покупки друзей

- Начисляются согласно условиям по уровням иерархии.
- До 3 уровней (покупки друга, друга друга, друга друга друга).

---

## SabyGet — пользовательский интерфейс

На карточке заведения покупатель видит:
- Кнопку **«Рекомендовать нас»** (если ещё нет приглашённых) или список приглашённых друзей
- Статистику: кол-во активных бонусов, количество принявших приглашение, итог по всей системе, разбивку по уровням
- При нажатии «Рекомендовать нас» — всплывает «Реферальная ссылка скопирована»
- Детализацию по конкретным друзьям (только 1-й уровень)

При переходе по реферальной ссылке — открывается карточка заведения (раздел «о нас») с кнопкой **«Присоединиться»**.

---

## Дополнительные правила

- При присоединении к программе в SabyGet покупатель может **автоматически получить дисконтную карту**, если она опубликована в аккаунте заведения.
- Реферальные бонусы начисляются именно на **персональную карту** покупателя.
- Расходуются бонусы по общим правилам для бонусных программ.
