# Wasaby Performance Budget

Система ограничений производительности для продуктов СБИС/Saby. Замеры проводятся инструментом **Perfalyze** в согласованных условиях.

> [!key-insight] Суть
> Каждому «типажу» (замеряемое действие × UI-объект × приложение) назначены пороги метрик. Превышение → ошибка в трекере.

---

## Метрики

| Метрика | Расшифровка |
|---|---|
| **VR** | Visual Readiness — время до первого значимого отображения (мс) |
| **TTI** | Time to Interactive — время до интерактивности (мс) |
| **REQ** | Количество сетевых запросов (шт) |
| **Size** | Объём переданных данных (КБ) |
| **BL** | Количество вызовов методов бизнес-логики (шт) |
| **Leak** | Утечка памяти (КБ) |
| **TRF** | Сетевой трафик при относительных замерах (КБ) |

---

## Относительные ограничения

Применяются к **любому типажу** — относительно предыдущего замера (или последней заведённой ошибки).

| VR | TTI | REQ | Size | TRF |
|---|---|---|---|---|
| +300 мс | +500 мс | +30 шт | +500 КБ | +150 КБ |

---

## Правила (не количественные)

- Устаревших модулей ≤ 100 шт
- DOM-нод ≤ 2000, глубина DOM ≤ 23
- Компонент ≤ 100 шт
- Неиспользуемый JS < 60%, CSS < 90%
- Нет запросов со статусом 3xx / 4xx / 5xx
- Тела ответов минифицированы
- HTTP/2 и выше для всех запросов
- Ответы >4 КБ сжаты gzip (кроме base64)
- Нет дублирующихся запросов при открытии страницы
- Нет `@import` в CSS, нет JS-кода в CSS
- Статика кэшируется >86400 сек; HTML — не кэшируется
- `window.componentOptions` < 50 КБ

---

## Исключения из бюджета

Если базовый бюджет объективно не достижим для конкретного объекта — оформляется **исключение** (согласованное переопределение). Может менять часть или все ограничения типажа.

Пример: страница с виджетом в браузере может иметь исключение VR=1800 (вместо базовых 1000) при наличии тяжёлого стороннего виджета.

## Инструменты

- **Perfalyze** (`test-perfalyze.sbis.ru`) — замеряет метрики; результаты сравниваются с бюджетом
- **explain.sbis.ru** — мониторинг PostgreSQL: проблемные запросы, блокировки, ошибки (только Тензор)
- **cloud.sbis.ru** — статистика и логирование методов БЛ
