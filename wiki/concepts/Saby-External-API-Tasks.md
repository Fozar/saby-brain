# Saby External API — СБИС.СписокЗадач

Метод возвращает список документов сотрудника из разделов **Задачи/На мне** и **Задачи/От меня**.

**Адрес запроса:** `https://online.sbis.ru/service/?srv=1`  
**Заголовок:** `X-SBISSessionID: <session_id>` (см. [[Saby-External-API-Auth]])

## Параметры запроса

| Поле | Тип | Обяз. | Описание |
|---|---|---|---|
| `Фильтр.ТипСписка` | строка | ✓ | `"НаМне"` или `"ОтМеня"` |
| `Фильтр.Сотрудник` | объект | — | Для запроса задач другого сотрудника. Нужны права. |
| `Фильтр.Сотрудник.Идентификатор` | строка | условно | ID сотрудника во внешней системе; если передан — ФИО игнорируется |
| `Фильтр.Тип` | строка | — | Тип документа (напр. `"ПриемНаРаботу"`) |
| `Фильтр.ИдентификаторДокумента` | строка | — | ID документа |
| `Фильтр.Регламент` | объект | — | `{Идентификатор, Название, Этап}` |
| `Фильтр.ДопПоля` | строка | — | `"ДополнительныеПоля"` — вернуть допполя документа |
| `Фильтр.Навигация.РазмерСтраницы` | строка | — | 1–50; число записей на страницу |
| `Фильтр.Навигация.Позиция.ДатаВремя` | строка | — | Курсор следующей страницы из предыдущего ответа |

## Пример запроса — «На мне», 10 записей

```json
{
  "jsonrpc": "2.0",
  "method": "СБИС.СписокЗадач",
  "params": {
    "Фильтр": {
      "ТипСписка": "НаМне",
      "Навигация": { "РазмерСтраницы": 10 }
    }
  },
  "id": 0
}
```

## Структура документа в ответе

```
Документ[]
  Идентификатор       — GUID документа в Saby
  Дата                — "ДД.ММ.ГГГГ"
  ДатаВремяСоздания   — "ДД.ММ.ГГГГ ЧЧ.ММ.СС"
  Номер               — строка
  Название            — строка
  Тип                 — тип документа (см. doc_guide)
  Направление         — напр. "Внутренний"
  Сумма               — строка
  Примечание          — строка
  Удален              — "Да" / "Нет"
  ЗакрытОтИзменений   — "Да" / "Нет"
  ОтметкаПлюсом       — "Да" / "Нет"
  Состояние
    Код                — код состояния (напр. "10" = "В обработке")
    Название
    Примечание         — комментарий при отклонении / текст ошибки
  Ответственный        — {Идентификатор, Фамилия, Имя, Отчество}
  Подразделение        — {Идентификатор, Название}
  Регламент            — {Идентификатор, Название}
  Редакция[]           — {Идентификатор, ДатаВремя, ПримечаниеИС, Актуален}
  Событие[]            — {ДатаВремя, Название, Идентификатор, Исполнитель, Комментарий, Подразделение}
  Срок                 — "ДД.ММ.ГГГГ"
  Расширение           — допполя (если ДопПоля="ДополнительныеПоля")
```

## Пагинация

Курсорная навигация по `Навигация.Позиция.ДатаВремя`:

```json
// Следующая страница — берём курсор из предыдущего ответа:
"Позиция": { "ДатаВремя": "1701787335.275692" }
```

В ответе: `Навигация.ЕстьЕще = "Да"/"Нет"` + `СледующаяПозиция.ДатаВремя` — курсор для следующего запроса.

## Доступ к задачам другого сотрудника

Требует прав просмотра. Передаётся через `Фильтр.Сотрудник`. Если указан `Идентификатор`, ФИО игнорируется — можно передавать без него.
