# Приложения семейства Version Control

Пять приложений, входящих в состав семейства Version Control и необходимых для работы Планов выпуска.

## Состав семейства

| Приложение | Назначение |
|-----------|-----------|
| **version-manager** | Непосредственно сам Хоттабыч |
| **version-manager-twin** | Приложение-близнец Хоттабыча; служит для обновления version-manager |
| **sbis-admin-repo** | Хранилище дистрибутивов, основанное на сервисе File Storage |
| **release-external** | Сервис интерфейса для «Выпуска версий», используемый в «Управлении версиями»; сервис БЛ для проксирования вызовов с «Управления версиями» на «Выпуск версий» |
| **release-manager** | Управление выпусками версий приложений; поддерживает ручной запуск отдельных работ и автоматическое выполнение работ по сценарию |

## Параметры работы с Выпуском версий

Параметры доступны в облаке по ссылке «Настройки».

Для поиска параметров, относящихся к планам выпуска, нужно в поисковой строке ввести:
```
управление версиями/планы выпуска
```

Будут отсортированы все параметры, связанные с планами выпуска в Хоттабыче.
