# SabyGet Loyalty Subsystems

Три подсистемы лояльности, доступные авторизованным пользователям Saby Get.

---

## Реестр «Карты» — sabyget.ru/cards

Хранение дисконтных карт: автоматически выданных при вступлении в бонусную программу + добавленных вручную сторонних.

> [!note] Виджет бонусной программы на странице заведения
> ![[2d808f74977b755163446d518de76de7_MD5.avif]]
> Страница заведения «Мили» (ресторан, рейтинг 4.6) показывает виджет: **«Кэшбэк до 10% + 300 Б в подарок!»** — карусель бонусных карт прямо на странице заведения. Так пользователь вступает в программу лояльности.

Реализован как «Легковесная иерархическая плитка».

На плитке карты: количество бонусов, название, фон, признак закреплённости.

### BL-методы

| Метод | Назначение |
|-------|-----------|
| `BonusCard.GetCardListShortFrm` | Список карт пользователя |
| `BonusCard.Read` | Детали карты в диалоге |
| `Establishment.GetLoyaltyProgram` | Программа лояльности заведения |
| `CardTemplate.GetList` | Шаблоны для создания новой карты |
| `BonusCard.Create` | Создание карты после выбора шаблона |
| `BonusCard.Save` | Сохранение отредактированной карты |

### Диалог карты (содержимое)

> [!note] Скриншот диалога карты
> ![[09a350bceaa7b944ccd75d6c0ec7c96b_MD5.avif]]
> Диалог карты: жёлтый фон, фото (кот), номер карты `Saby0000000000`, штрихкод. Ниже — список заведений где можно применить («Saby продаёт рыбу»), счётчик «5 бонусов», секция **«Получайте до 5% накоплений у друзей»** с кнопкой «Рекомендовать нас» — реферальная программа.

1. Закрепление/открепление
2. Количество бонусов
3. Логотип организации
4. Название карты
5. Настроенное изображение
6. Уникальный код карты
7. QR-код для мобильного
8. Список заведений для применения
9. Список промоакций владельца
10. Реферальная программа

**Действия по типу карты:** Закрепить/Открепить | Удалить | Изменить (только для добавленных вручную).

## Реестр «Бонусы» — sabyget.ru/bonuses

Бонусные карты в разрезе заведений + история бонусов за последний год.

> [!note] Скриншот страницы «Бонусы»
> ![[a5d92e4f7f1132e79f5a56b8fad12e32_MD5.avif]]
> Шапка: **«У вас 424 Б»**. Карусель: Ресторан Мили (120 Б), Верблюж (2 120 Б), Loft. Итого: **30 424 Б начислено за последний год** — из них 21 560 с покупок, 8 864 от друзей. Список: Ресторан Мили 5 040, Мамука 1 000, Клюква 440, Морошка 25... Справа — карта Ярославля с отмеченными заведениями.

**Компоненты страницы:**
- Карусель с кратким списком бонусных карт (по клику «Все» → полный реестр)
- Список бонусов за последний год (3 счётчика: всего / с покупок / от друзей)
- Диалог детализации (из карусели или из списка)

**Диалог детализации содержит:**
1. Название заведения
2. Количество бонусов
3. Выгода реферальной программы
4. Кнопка «Поделиться реферальной ссылкой»
5. Бонусы за последний год
6. Ступени выгод реферальной программы
7. Список покупок

### BL-методы

| Метод | Назначение |
|-------|-----------|
| `BonusCard.CardListWithBonuses` | Краткий и полный список карт с бонусами |
| `BonusOperation.PersonAccruedBonuses` (GetAccruedBonuses) | Список бонусов за последний год |
| `Establishment.BonusDetails` | Детализация начислений по заведению |
| `ShowcaseOrder.ListForShowcase` | Список покупок в диалоге |

> [!key-insight] Реферальная программа в Бонусах
> Диалог детализации бонусов в SabyGet — основная точка входа в реферальную программу для пользователей. Здесь видны ступени выгод и кнопка «Поделиться ссылкой». Связанная страница: [[ReferralDeals-System]].

## Реестр «Промокоды» — sabyget.ru/promocodes

Промокоды пользователя, полученные за: написание отзыва, прохождение опроса, оформление заказа, присоединение к программе лояльности.

> [!note] Скриншот страницы «Промокоды»
> ![[b6620e45b36d076d4bb424a724e821e6_MD5.avif]]
> Реестр промокодов в виде плиток с фото хлеба/выпечки. Коды: AL2K7, 6ZABTB, BXOMS. Справа — карта Ярославля. Реестр пустой у большинства пользователей — промокоды выдаются за конкретные действия.

Реализован как «Легковесная иерархическая плитка».

На плитке: код промокода, фон, описание.

**Диалог промокода содержит:** название заведения, код, выгоду, QR-код + (опционально) список заведений для применения.

### BL-методы

| Метод | Назначение |
|-------|-----------|
| `SGPromocode.GetList` | Список промокодов пользователя |
| `Promocode.Get` | Данные конкретного промокода |
| `PromocodeType.ReadAspectSettings` | Настройки аспекта промо-страницы |
| `ShowcasePublic.TryGetEstLinkBySalePoint` | Список заведений для применения |

## Связи с core-системой лояльности

```
SabyGet UI ←→ BL-методы (BonusCard.*, SGPromocode.*, Promocode.*)
                    ↓
        DiscountCard-сервис (Карта, КартаШаблон)
        Promocode-сервис (Промокод, ПромокодТип)
        Bonus-сервис (BonusOperation, ВидЦены)
```

Карты, видимые в SabyGet — те же объекты `Карта` системы лояльности СБИС (см. [[DiscountCard-Subsystem-Overview]]). Промокоды — те же `Промокод`/`ПромокодТип` (см. [[Promocode-Subsystem-Overview]]).
