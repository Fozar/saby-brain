# Wasaby — Настройка локального стенда

## Файловая структура

```
Saby/
└── local_stand/
    ├── Project/    # файлы проекта (дистрибутив распаковывается сюда)
    └── Server/     # запуск локального стенда
```

> [!note] Windows
> Рекомендуемые пути: `C:\Saby`, `D:\Saby`, `E:\Saby` — добавлены в исключения антивируса Kaspersky (корпоративная политика).

## Тестовые стенды

| Домен | Описание |
|-------|----------|
| `test-online.sbis.ru` | Ближайший к релизу дистрибутив |
| `pre-test-online.sbis.ru` | Следующий за готовящимся к выпуску |
| `dev-online.sbis.ru` | Trunk (далеко от релиза) |
| `pre-dev-online.sbis.ru` | Ещё дальше от релиза (доступ по поручению) |
| `test-online.setty.kz` | Казахстан |
| `test-online.saby.uz` | Узбекистан |

### Логины для стендов

| Стенд | Логин / Пароль |
|-------|----------------|
| test-online / pre-test-online | `создатель_сотрудников` / `создатель_сотрудников123` |
| dev-online | `Демо` / `Демо123` |
| test-online.setty.kz | `Создатель_сотрудников_KZ` / `Создатель_сотрудников_KZ123` |
| test-online.saby.uz | `Создатель_сотрудников_UZ` / `Создатель_сотрудников_UZ123` |

Версию стенда: `<домен>/ver.html` → поле `ver`.

## Получение дистрибутива

Из сетевого диска:
```
# Россия (Windows):
\\sbis-dev.corp.tensor.ru\programmers\СБиС_3\Локальный стенд\test\<версия>\win\

# Россия (Linux):
smb://sbis-dev.corp.tensor.ru/programmers/СБиС_3/Локальный стенд/test/<версия>/nix/

# Другие страны (Windows):
\\sbis-dev.corp.tensor.ru\programmers\СБиС_3\Тестирование\online-inside_<версия>.ls\win\
```

Условно-стабильные (нетестированные) дистрибутивы:
```
\\sbis-dev.corp.tensor.ru\programmers\СБиС_3\Тестирование\<дистрибутив_версия>\
```

## Установка и запуск в Genie

1. Распаковать дистрибутив в `Project/` — **использовать 7-Zip** (стандартные средства зависают)
2. В [Genie](https://genie.saby.ru/) открыть файл дистрибутива из `Project/` (фильтр: "Cloud file")
3. Проверить версию SDK (должна совпадать со стендом) и открытый проект
4. Если нужен симлинк — создать **до** развертывания стенда

## Совместимость SDK с дистрибутивом

Из файла `Схема сборки дистрибутива.s3distr` в корне дистрибутива:
```xml
version_sdk="26.2000"     <!-- совместимая версия SBIS SDK -->
build_date="10.03.2026"
build_number_sdk="311"
```

## Выбор БД при развертывании

| Тип БД | Когда использовать |
|--------|-------------------|
| Локальная | Активная модификация схемы БД, нужна конвертация |
| Облачная | Исправление бизнес-логики, разработка статики |

> [!warning] Облачная БД
> После изменений — проверить облачный (не локальный) стенд: отвязать пользователя от стенда в регистраторе и авторизоваться на `test/pre-test.online.sbis.ru`.

## Watcher и симлинки

Watcher — компонент, отслеживающий изменения из репозитория. Если симлинки созданы после развертывания — нужен **редеплой** для обновления.
