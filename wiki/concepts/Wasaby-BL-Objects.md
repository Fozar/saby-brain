# Wasaby BL Objects

БЛ-объекты (Business Logic Objects) — основная единица бизнес-логики в Wasaby. Организованы в иерархию наследования.

## Иерархия объектов

```
БазовыйОбъект
├── ДоменныйОбъект1          ← домен (например, "Документ")
│   ├── КонкретныйОбъект1    ← конкретная сущность
│   └── КонкретныйОбъект2
└── ДоменныйОбъект2
    └── КонкретныйОбъект3
```

- `БазовыйОбъект` определяет базовые методы (CRUD), которые наследуются автоматически
- Дочерние объекты могут переопределять родительские методы
- Глубина наследования не ограничена, но рекомендуется не более 3-4 уровней

## Типы методов

| Тип | Описание | Пример |
|-----|----------|--------|
| **Авто-CRUD** | Генерируются фреймворком по схеме объекта | `Read`, `Create`, `Update`, `Delete` |
| **List** | Возвращает список записей с фильтрацией/пагинацией | `List`, `ListGetPage` |
| **File** | Работа с файлами / бинарными данными | `GetFileList`, `Upload` |
| **Remote** | Метод, доступный для вызова из другого сервиса | задекларирован в контракте |
| **Локальный** | Вызывается только внутри сервиса | нет в контракте |

## Наследование и переопределение методов

Дочерний объект наследует все методы родителя. Метод может быть:

1. **Унаследован** — используется без изменений
2. **Расширен** — дочерний добавляет логику до/после вызова родителя
3. **Переопределён** — дочерний полностью заменяет реализацию родителя

```python
class ChildObject(ParentObject):
    def Read(self, params):
        # расширение: преобразовать параметры
        params['extra'] = True
        result = super().Read(params)
        # расширение: обогатить результат
        result['augmented'] = True
        return result
```

## Исключение прав (Is always permitted)

Флаг `Is always permitted` на методе отключает стандартную проверку прав доступа для этого метода. Используется для:

- Системных методов, вызываемых сервисными процессами без пользовательского контекста
- Методов межсервисного взаимодействия с политикой `AuthByApplicationId`
- Методов инициализации/миграции данных

> [!key-insight] Флаг опасен
> `Is always permitted` полностью обходит систему прав. Устанавливать только на методы, которые изолированы от прямого клиентского вызова или проверяют аутентификацию самостоятельно.

## Регистрация объекта

Объект регистрируется по имени вида `'Домен.Объект'` — этот же идентификатор используется при вызове через `BLObject(...)`.
