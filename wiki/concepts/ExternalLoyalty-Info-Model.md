# External Loyalty — Information Model

Информационная модель подсистемы внешней лояльности (UDS, PremiumBonus, iikoCard).

---

## Термины

| Термин | Описание |
|--------|----------|
| Внешняя Бонусная Система | Объект, описывающий сторонний программный комплекс для ведения клиентской базы и персональной лояльности |
| Продажа | Факт совершённой кассиром продажи товаров/услуг |
| Применение Карты | Факт применения КартыЛояльности к продаже |
| Применение Скидки | Факт применения объекта скидки к продаже |
| Применение Бонусов | Факт применения бонусов к продаже |
| Возврат Продажи | Факт возврата продажи |

---

## Ключевой объект: ВнешняяБонуснаяСистема

Методы интерфейса:

| Метод (рус) | Метод (eng) | Аргументы | Возвращает | Описание |
|-------------|-------------|-----------|-----------|----------|
| ПоискКлиента | `FindClient` | `Идентификатор: str` | `ФизЛицо` | Поиск клиента во внешней системе по произвольному идентификатору (карта/телефон/временный код) |
| РасчетЛояльности | `GetExternalLoyalty` | `ПрименениеКарты` | `ПрименениеСкидки`, `ПрименениеБонусов` | Расчёт внешней лояльности по текущей продаже для найденного клиента |
| ПровестиПродажу | `ProcessSale` | `ПрименениеКарты`, `ПрименениеСкидки`, `ПрименениеБонусов` | `UUID внешней транзакции` | Фиксирует во внешней системе факт проведения продажи |
| ОтменаПроведенияПродажи | `ProcessCancelSale` | `ВозвратПродажи`, `ПрименениеКарты`, `ПрименениеСкидки`, `ПрименениеБонусов` | — | Фиксирует отмену ранее проведённой продажи |

### Внешние объекты (из других подсистем)

- **ФизЛицо** — прикладной объект из CRM
- **Продажа** — прикладной объект из подсистемы продаж
- **ВозвратПродажи** — прикладной объект из подсистемы возвратов

> [!note]
> На участке iikoCard отсутствуют прикладные объекты, клиентские реестры/панели или печатные формы — интеграция полностью прозрачна для интерфейса.

## Схема связи объектов

```
ВнешняяБонуснаяСистема
  ├── FindClient(id) → ФизЛицо
  ├── GetExternalLoyalty(ПрименениеКарты) → ПрименениеСкидки + ПрименениеБонусов
  ├── ProcessSale(Карта + Скидка + Бонусы) → ExternalTransaction UUID
  └── ProcessCancelSale(Возврат + Карта + Скидка + Бонусы) → void

ПрименениеКарты
  ├── Sale: Продажа
  ├── Card: КартаЛояльности → Владелец: ФизЛицо
  │                         └── Эмиссия → Акция / БонуснаяПрограмма
  ├── BonusDecLimit: Decimal
  └── ExternalTransaction: UUID | None
```
