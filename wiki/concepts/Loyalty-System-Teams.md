# Система лояльности — Структура направления

Направление **Система лояльности** входит в структуру [[Tensor-Company]] (подразделение: Разработка → Склад и госсистемы). Существует вместе с [[Transport-VIS-Platform]] под общим руководством [[Мусохранов-Андрей-Владиславович|Мусохранова А.В.]].

## Org Chart

```
Мусохранов А.В. (Руководитель направления 2+)
├── Система лояльности
│   └── Федько Ю.С. (Руководитель направления 2)
│       ├── Акции и бонусы, механика скидок (backend) → Алябушев А.А. (3+)
│       ├── Дисконтные карты, лояльность на продаже (backend) → Омельяненко Е.А. (3+)
│       ├── Система лояльности (frontend) → Курников М.С. (3+)
│       └── QA: Система лояльности → Дюднева С.А. (3+)
└── Транспорт ВИС → Чумакин А.А. (2) [см. [[Transport-VIS-Platform]]]
```

## Команды

### Акции и бонусы, механика скидок (backend)

**Руководитель:** [[Алябушев-Александр-Александрович]] (3+)

- Механика расчёта скидок
- Серверная часть справочника Скидок, Бонусов, Подарков
- Серверная часть импорта дисконтных карт
- API сторонним командам
- Отображение карт в SabyGet
- Выпуск электронной карты через онлайн-анкету

Концепции: [[Акции-Subsystem-Overview]], [[Акции-Architecture]], [[DiscountCard-Service-API]]

### Дисконтные карты, лояльность на продаже (backend)

**Руководитель:** [[Омельяненко-Егор-Анатольевич]] (3+)

- Применение/отображение скидок, бонусов, дисконтных карт на продаже (розница, склад)
- Редактирование/отображение списка карт, бонусов, скидок в карточке клиента
- Справочник сертификатов, их применение и продажа в рознице

Концепции: [[DiscountCard-Subsystem-Overview]], [[DiscountCard-Algorithms-Processes]], [[DiscountCard-Admin-Ops]]

### Система лояльности (frontend)

**Руководитель:** [[Курников-Михаил-Сергеевич]] (3+)

- Клиентская часть работы с ценами, прайсами/меню, документами изменения цен
- Карточка и реестр бонусных программ
- Админка сервисов: «Шаблоны нумераций», «Ограничения цен», «Очереди задач»

Концепции: [[Акции-UI]], [[DiscountCard-UI-Specifics]]

### QA: Система лояльности

**Руководитель:** [[Дюднева-Светлана-Андреевна]] (3+) | Куратор: Маренин И.

Функциональное тестирование: прайсы, скидки, акции, дисконтные карты, сертификаты, документ изменения цен, сервис ограничения цен.

## Продукты и интеграции

Система лояльности обслуживает пользователей продуктов: **Розница**, **HoReCa**, **SabyGet** (включая автономные аналоги).

Внешние интеграции: [[ExternalLoyalty-Integrations]] — UDS, PremimBonus, iikoCard и другие.
