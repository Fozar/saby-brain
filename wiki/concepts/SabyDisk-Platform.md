# Платформа SabyDisk / Saby Space

Единая платформа для хранения и работы с пользовательскими файлами в СБИС.

## Назначение

- Заменяет СУБД для хранения бинарных пользовательских данных (дешевле)
- Единая точка для всех приложений (ЭДО, Отчётность, Новости, Переписка и т.д.)
- Потоковый API (экономия времени загрузки/скачивания)
- ГОСТ-криптография и электронные подписи
- Шаринг, версионирование, иерархия каталогов, контроль доступа

## Продукты

| Продукт | Описание |
|---|---|
| **Saby Space** (space.saby.ru) | Единое пространство пользователя (аналог Google Drive) |
| **МП Saby Space** | Мобильное приложение |
| **Платформа SabyDisk** | Внутренняя инфраструктура для разработчиков |

## Основные принципы

1. **Бинарные данные** → SWIFT/CEPH объектные хранилища (дёшево)
2. **Метаданные** → RPC API из любой точки системы (с кэшированием)
3. **Потоковый доступ** → REST API (скачивание частями, видео)

## Декомпозиция сервисов

| Сервис | Назначение |
|---|---|
| `diskrest` | REST API (потоковая загрузка/скачивание), сайт `disk.sbis.ru` |
| `diskrest-public` | Копия для неавторизованных (DDoS-защита) |
| `diskrest-internal` | Только из облака, без проверки прав |
| `СБИСДиск — корневой` | Корни и квоты; все сервисы ходят сюда |
| `СБИСДиск — основной` | RPC API; метаданные объектов |
| `СБИСДиск — main service public` | Копия для неавторизованных |
| `СБИСДиск — базовый` | Облегчённый функционал |
| `СБИСДиск — приватный` | Для внутренних вызовов (права уже проверены) |
| `СБИСДиск — длительные операции` | Тяжёлые операции (крипто-хэш >1 ГБ) |
| `СБИСДиск — служебный` | Синхронизация ссылок, реиндекс, отложенное удаление |
| `disk-elasticsearch` | Поиск |
| `disk-search` | Обработка листинговых запросов с поиском |
| `SbisDisk - binary data service` | Перенос бинарных данных из "горячего" кэша в SWIFT/CEPH |
| `SbisDisk - data utility service` | Тяжёлые служебные задачи (массовые read/write объектов) |

## Специальные решения хранения

- **Составные файлы версий** — XML-документы >100 МБ → несколько независимых бинарных объектов
- **Служебные файлы** — обложка, логотип, виджеты папки (скрыты при обычном листинге)
- **Циклическая запись** — включается на пространстве; при переполнении квоты удаляются старые файлы (файлы хранятся плоским списком, папки запрещены); применяется для Звонков

## Внешние зависимости СБИСДиска (основной)

- `history-disk` — события истории
- `right-mng` — ограничения на участок
- Сервис профилей — данные персон
- Online — проверка внешних прав

## Информационная модель

`disk.sbis.ru` — выделенный сайт для download/upload (ограничение полосы по клиенту на физическом уровне).
