# Saby Naming Guide

Navigation: [[index]] | [[Tensor-Company]] | [[Tensor-Glossary]]

Официальное руководство по написанию названий продуктов Saby (версия Jan 2025).

## Правила написания

> [!key-insight] Главное правило
> "Saby" — всегда латиницей. Первая буква заглавная, остальные строчные. Мужского рода в русском языке.

### Составные названия продуктов
Формат: `Saby <Слово>` — оба слова через пробел, второе слово с заглавной буквы.

```
✅ Saby Retail
✅ Saby Report
❌ SABY RETAIL
❌ saby retail
```

### Запрет смешивания языков
```
❌ Подключили Saby Розницу
❌ Внедрили Saby КЭДО
✅ Подключили Saby Retail
✅ Внедрили Saby Staff / Автоматизировали магазин в Saby
```

### Продукты без собственного приложения
Если у продукта нет своего приложения — используем просто "Saby".
Пример: СБИС Маркировка → Saby.
