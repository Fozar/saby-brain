# BonusDecRule — Правило Списания Бонусов

Прикладной объект. ВидЦены с Тип=40 (`BONUS_DEC_RULE`). Каждое правило списания бонусов неявно ограничено **глобальными настройками списания** (действуют как глобальный ограничитель номенклатур).

---

## ПравилоСписанияБонусов (BonusDecRule)

| Название | Eng | Тип | Описание |
|----------|-----|-----|----------|
| Название | Name | String | Название правила |
| Выгода | Benefit | Benefit | Величина и список номенклатур |
| УсловияСрабатывания | TriggerConditions | TriggerConditions | Когда, где и для кого активируется |

---

## Выгода (Benefit)

| Название | Eng | Тип | Описание |
|----------|-----|-----|----------|
| Величина | Value | Decimal | Процент списания бонусов |
| ПороговыеУсловия | ThresholdConditions | ThresholdConditions | Пороги для процента списания |
| Номенклатуры | Nomenclatures | Nomenclature[] | На какие товары действует |
| ИсключенныеНоменклатуры | ExcludedNomenclatures | Nomenclature[] | На какие не действует |
| ТипПрименения | ApplicationType | Enum | — |

**ТипПрименения (ApplicationType)**:
- `0` — на номенклатуры из Номенклатуры, или на все кроме ИсключенныхНоменклатур
- `1` — на все номенклатуры продажи
- `2` — на номенклатуры из УсловияСрабатывания.Номенклатуры
- `3` — на самую дешёвую номенклатуру в продаже
- `4` — на самую дешёвую из Номенклатуры (или из всех, кроме Исключённых)

---

## ПороговыеУсловия (ThresholdConditions)

| Название | Eng | Тип | Описание |
|----------|-----|-----|----------|
| Тип | Type | Enum | Вид порогового условия |
| ЗначениеПериода | PeriodValue | Integer | Период (в ед. ТипПериода) |
| ТипПериода | PeriodType | Enum | За период / за неделю / месяц / квартал / весь период |
| ЕдиницаИзмеренияПериода | PeriodUnit | Enum | День / неделя / месяц |
| Пороги | Thresholds | Threshold[] | Список пороговых сумм и процентов |

**Тип порогового условия**:
- `0` — сумма продажи
- `1` — количество любых товаров в продаже
- `2` — общая сумма по всем продажам
- `3` — общее количество любых товаров по всем продажам
- `4` — количество одной конкретной номенклатуры в продаже

**ТипПериода**:
- `0` — за период (нужно ЗначениеПериода)
- `1` — за последнюю неделю
- `2` — за последний месяц
- `3` — за последний квартал
- `4` — за весь период

---

## Порог (Threshold)

| Название | Eng | Тип | Описание |
|----------|-----|-----|----------|
| Лимит | Limit | Decimal | Величина порога для срабатывания |
| Значение | Value | Decimal | Процент списания при достижении порога |

## УсловияСрабатывания (TriggerConditions)

| Название | Eng | Тип | Описание |
|----------|-----|-----|----------|
| Номенклатуры | Nomenclatures | Nomenclature[] | Обязательные товары для срабатывания |
| ПериодСрабатывания | TriggerPeriod | TriggerPeriod | Даты, время, дни недели/месяца |
| ТочкиПродаж | SalesPoints | SalesPoint[] | Точки продаж |
| ЗалыСтолы | HallsTables | HallTable[] | Залы/столы заведения |
| НаДоставку | ForDelivery | Enum | 0=нет, 1=самовывоз, 2=курьером, 3=оба |
| ВДеньРождения | OnBirthday | Integer | Дней до/после ДР клиента |
| ДисконтныеКарты | DiscountCards | LoyaltyCard[] | При наличии карты выбранного типа |
| Клиенты | Clients | Client[] | Для выбранных клиентов |
| ИсключенныеКлиенты | ExcludedClients | Client[] | Не для выбранных клиентов |
| ПервыйЗаказ | FirstOrder | Boolean | За первый заказ |
| Регионы | Regions | Region[] | В выбранных регионах |
| ИсключенныеРегионы | ExcludedRegions | Region[] | Не в выбранных регионах |

## БД представление

```
Глобальные настройки списания — ВидЦены (Type=?), глобальный ограничитель
ПравилоСписанияБонусов        — ВидЦены (Type=40)
  ВидЦеныНоменклатура         — перечень номенклатур
  ВидЦеныЭмиссия              — перечень эмиссий карт
  ВидЦеныЛица                 — персональные настройки (Лицо + ПапкаЛицо)
  <Сервис параметров>          — максимальный % оплаты бонусов, флаг начисления при списании
```

> [!key-insight] Глобальные настройки как ограничитель
> Глобальные настройки списания ограничивают целевые номенклатуры во всех правилах. Каждое правило дополняется запрещёнными номенклатурами из глобальных настроек.
