# Акции — Информационная модель

Navigation: [[index]] | [[Акции-Subsystem-Overview]] | [[Акции-Architecture]]

## Иерархия объектов

```
Акция (Promotion)
└── Скидка (Discount)          — расширяет Акцию
    └── УсловияСкидки          — расширяет УсловияАкции
```

Вспомогательные типы: `УсловияАкции`, `Выгода`, `ПероходныеУсловия`, `Порог`, `ПериодСрабатывания`

Внешние объекты (не Акции): `Промокод`, `Подсказка`, `ПромоСтраница`, `КартаЛояльности`

## Акция (Promotion)

Прикладной объект.

| Поле | EN | Тип | Описание |
|------|----|-----|----------|
| Название | Name | String | Название Акции |
| Используется | IsActive | Boolean | Является ли Акция действующей |
| Условия | Conditions | УсловияАкции | При каких критериях предоставляется Выгода |
| Подсказка | Prompt | Подсказка | Связанная Подсказка |
| Реклама | Advertising | Реклама | Настройки рекламы |
| ПромоСтраница | PromoPage | ПромоСтраница | Публичная страница с информацией |
| Описание | Description | String | Описание Акции |
| Изображение | Image | String | ID картинки на СбисДиске (баннер карусели) |
| ИзображениеДляЭкрана | ScreenImage | String | ID картинки для экрана покупателя |

## Скидка (Discount)

Расширяет Акцию.

| Поле | EN | Тип | Описание |
|------|----|-----|----------|
| Выгода | Profit | Выгода | На что распространяется скидка |
| Условия | Conditions | УсловияСкидки | Критерии для получения скидки |
| Вручную | IsManual | Boolean | Назначается ли вручную кассиром |
| СписокРолей | RoleList | Роль[] | Сотрудники с какой ролью могут применить |

## УсловияАкции (PromotionConditions)

| Поле | EN | Тип | Описание |
|------|----|-----|----------|
| Клиенты | Clients | Клиент[] | Акция для выбранных клиентов |
| ИсключенныеКлиенты | ExcludedClients | Клиент[] | Акция не для выбранных клиентов |
| ДисконтныеКарты | DiscountCards | Эмиссия[] | Акция при наличии карты выбранного типа |
| ВДеньРождения | OnBirthday | Integer | Акция за N дней до/после дня рождения |
| ПервыйЗаказ | FirstOrder | Boolean | Акция за первый заказ |
| Регионы | Regions | Регион[] | Применяется в выбранных регионах |
| ИсключенныеРегионы | ExcludedRegions | Регион[] | |
| Промокод | PromoCode | Промокод | При указании промокода |
| ПериодСрабатывания | Period | ПериодСрабатывания | Период, время, дни недели/месяца |
| ТочкиПродаж | SalesPoints | ТочкаПродаж[] | В выбранных точках продаж |
| НаДоставку | ForDelivery | Enum | 0=не действует / 1=самовывоз / 2=курьер / 3=оба |
| ВидОплаты | PaymentType | Integer[] | 1=нал / 2=картой / 3=зарплата / 4=QR / 5=интернет |
| ИсточникПродажи | SaleSources | ИсточникПродажи[] | Источники продажи |
| ЗалыСтолы | HallsTables | ЗалСтол[] | Залы или столы заведения |
| Номенклатуры | Nomenclature | Номенклатура[] | При покупке выбранных товаров |

## УсловияСкидки (DiscountConditions)

Расширяет УсловияАкции.

| Поле | EN | Тип | Описание |
|------|----|-----|----------|
| ТипПродажи | PayType | Клиент[] | 0=оптовая / 1=розничная |
| НоменклатурыКоличество | NomenclatureCount | Клиент[] | При покупке товаров в выбранном кол-ве |
| СуммаПродажи | SaleSum | Decimal | При покупке на выбранную сумму |
| Суммируется | IsSummable | Boolean | Суммируется с другими скидками |
| СрабатываетСоСкидками | WithDiscounts | Скидка[] | Применяется совместно с указанными скидками |
| ПриоритетПрименения | Priority | Enum | -1=низкий / 0=средний / 1=высокий |

## Выгода (Profit)

| Поле | EN | Тип | Описание |
|------|----|-----|----------|
| Величина | Value | Decimal | Сумма скидки (без пороговых условий) |
| ПороговыеУсловия | ThresholdConditions | ПороговыеУсловия | Диапазон значений в зависимости от условий |
| Номенклатуры | Nomenclatures | Номенклатура[] | На что действует выгода |
| ИсключенныеНоменклатуры | ExcludedNomenclatures | Номенклатура[] | |
| ТипПрименения | ApplyType | Enum | 0=по списку/все кроме исключённых; 1=все; 2=из условий срабатывания; 3=самое дешёвое; 4=самое дешёвое из списка |
| ТипВеличины | UnitType | Enum | 0=процентная / 1=суммовая |

## ПероходныеУсловия (ThresholdConditions)

| Поле | EN | Тип | Описание |
|------|----|-----|----------|
| ТипПорога | ThresholdType | Enum | 0=сумма чека / 1=кол-во всех товаров / 2=сумма за период / 3=кол-во посещений / 4=кол-во одного товара |
| ЗначениеПериода | PeriodValue | Integer | Значение периода |
| ТипПериода | PeriodType | Enum | 0=за период / 1=за нед. / 2=за мес. / 3=за квартал / 4=за весь период |
| ЕдиницаИзмеренияПериода | PeriodUnit | Enum | 0=день / 1=неделя / 2=месяц |
| Пороги | Thresholds | Порог[] | Набор пар: условие → значение скидки |

## ПериодСрабатывания (Period)

| Поле | EN | Тип |
|------|----|-----|
| ДатаНачала | StartDate | Date |
| ДатаКонца | EndDate | Date |
| ВремяНачала | StartTime | Time |
| ВремяКонца | EndTime | Time |
| ДниНедели | DaysOfWeek | Integer |
| ДниМесяца | DaysOfMonth | Integer |
