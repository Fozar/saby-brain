# Saby API — Справочник кодов ошибок

Все ошибки API Saby приходят в JSON-RPC формате: `error.code = -32000`, `error.data.classid` — расширенный классификатор.

## Формат расширенного кода

```
00000000-0000-0000-0000-1XXXXXXXXXXX
```

- `1FA...` — **фатальные** (нужно исправить параметры запроса)
- `1AA...` — **нефатальные** (достаточно повторить запрос без изменений)
- `100000000000` — ошибка общего вида без классификации; как правило, фатальная

## Фатальные ошибки (`1FA...`)

| classid (суффикс) | HTTP | Описание |
|---|---|---|
| `1FA00001XXXX` | 500 | Ошибки входных параметров |
| `1FA000010002` | 500 | Ошибка адресации связанного документа |
| `1FA000010004` | 500 | Неполная/неверная информация о сертификате |
| `1FA000010005` | 500 | Неверный или несуществующий тип документа |
| `1FA000010007` | 500 | Ошибка адресации документа или редакции |
| `1FA00002XXXX` | 500 | Ошибки криптографии |
| `1FA00003XXXX` | 500 | Ошибки формализованных документов |
| `1FA00004XXXX` | 500 | Ошибки запуска в документооборот |
| `1FA00008XXXX` | 500 | Ошибки поиска абонентского ящика |
| `1FA0000DXXXX` | 500 | Ошибки лицензирования |
| `1FA0000FXXXX` | 500 | Ошибки форматно-логического контроля |
| `1FA0000A0001` | 500 | «Невозможно выполнить преобразование» (повреждённое вложение) |
| **`1FA000001000`** | **429** | **Rate limit**: ≥300 вызовов/мин → блокировка на 600 сек |
| **`1FA000001001`** | **500** | **Неверный логин/пароль** (или пустые поля) |
| **`1FA000001002`** | **500** | **Требуется SMS-подтверждение** (2FA) |

## Нефатальные ошибки (`1AA...`)

| classid (суффикс) | Описание |
|---|---|
| `1AA00001XXXX` | Ошибки внутренних сервисов |
| `1AA0000F1002` | Повторить запрос через 3–5 сек (асинхронный результат ещё не готов) |
| `1AA000001000` | Нефатальная ошибка аутентификации — повторить |

> [!note] Правило классификации
> Любой classid, не попадающий в таблицу `1AA...`, считается **фатальным**.

## Типовая обработка в клиенте

```python
def call_api(method, params, session_id):
    resp = post_json_rpc(method, params, session_id)
    if "error" in resp:
        classid = resp["error"]["data"]["classid"]
        if "1AA0000F1002" in classid:
            time.sleep(4)
            return call_api(method, params, session_id)  # retry
        if "1AA" in classid:
            return call_api(method, params, session_id)  # retry
        # fatal — raise
        raise SabyAPIError(resp["error"])
    return resp["result"]
```

При HTTP 401 — повторить [[Saby-External-API-Auth|аутентификацию]] и повторить запрос.
