# SabyGet Product Overview

Saby Get (`sabyget.ru`) — агрегатор заведений (рестораны, магазины, салоны красоты), позволяющий конечным пользователям взаимодействовать с бизнесом через единое приложение/сайт.

## Основные возможности

| Функция | Описание |
|---------|----------|
| Поиск и просмотр | Рестораны, магазины, салоны — цены, отзывы, ассортимент |
| Бронирование/запись | Столики в ресторанах, услуги в салонах — выбор специалиста/даты |
| Доставка | Оформление, оплата, отслеживание курьера на карте |
| Оплата по QR | Сканирование QR или ссылка + чаевые + отзыв после оплаты |
| Сканирование чеков | Реестр покупок, аналитика расходов |
| Дисконтные карты | Хранение карт заведений и сторонних скидочных карт |
| Бонусы и скидки | Просмотр и использование бонусов, промокодов, акций |
| Сертификаты | Покупка, использование, подарок другим пользователям |
| Избранное | Подписка на новости и уведомления от заведений |
| Коммуникация | Отзывы, чат с заведениями |
| Уведомления | Статусы заказов, бонусы, чаты, звонки от заведений |

## Аудитория

- **Пользователи (клиенты)**: конечные потребители — хотят быстрый доступ к информации, заказам, бронированиям
- **Владельцы бизнеса**: клиенты Saby Get — управляют страницей заведения, ассортиментом, программами лояльности

## Платформы

- Основной сайт: [sabyget.ru](https://sabyget.ru/)
- Мобильное приложение Saby Get
- Админка Saby Get
- Брендированные мобильные приложения
- Клиентские внешние виджеты
- Онлайн-формы

## Разделы сайта (аккордеон)

**Для гостя:**
- Разводящая страница, Новости, Карты, Покупки, Избранное, Чаты, Отзывы, Профиль

**Для авторизованного пользователя** дополнительно:
- Промокоды, Бонусы

**Для владельцев бизнеса** дополнительно:
- Мой бизнес (переход в Saby-приложение)

**Страницы по ссылкам (не из аккордеона):**
- Страница заведения, Страница курса, Страница события, Подборки по категориям, Сертификат

**Страницы на домене sabyget.ru (не из аккордеона):**
- Электронное меню, Сайт доставки, Сайт услуг, Сайт отеля, Сайт франшизы, Чек/счёт

## Основной принцип работы

Saby Get получает данные из сервисов **proxy-вызовами** по:
- аккаунту заведения из `online.sbis.ru`
- идентификатору группы соц. сети
- идентификатору пользователя в облаке
- идентификатору СПП организации

## SEO и кеширование

Индексируются: Разводящая страница, Страница заведения, Страница курса, Страница события/мероприятия, Сайт доставки, Сайт услуг, Сайт отеля, Сайт франшизы.

На большинстве индексируемых страниц настроено **Varnish-кеширование**.
