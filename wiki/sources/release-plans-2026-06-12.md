# Планы выпуска — документация (7 PDF)

## Источники

| Файл | Краткое содержание |
|------|--------------------|
| `Зависимости и запуск работ.pdf` | Модель зависимостей (4 типа), правила запуска этапов и отдельных работ, приоритеты, пропуск зависимостей, работа в региональных облаках |
| `Статусы работ.pdf` | Два вида статусов (автоматические Хоттабыча / пользовательские), списки статусов, влияние на выгрузку дистрибутивов через «покебол» |
| `Принципы формирования плана.pdf` | Регламент релиз-менеджеров: папки «на -» / «на -», ВНР/ОС, работа с мажорными выпусками, обязанности дежурного выпускающего |
| `Отчет по времени.pdf` | Отчёт по времени: графический вид + табличная часть, фильтры, проблемы обновления, форма заполнения инцидента |
| `Фильтры в текущем плане.pdf` | Фильтры реестра: по стенду, статусу, запрету, состоянию, типу работы; «Критичные», «Расхождение версий», «Присутствующие/отсутствующие в облаке» |
| `Приложения, входящие в состав семейства Version Control...pdf` | Семейство Version Control: version-manager, version-manager-twin, sbis-admin-repo, release-external, release-manager |
| `Техническая документация/Техническая документация.pdf` | Полное ТД: концептуальное решение, карточка плана, теги, Telegram-бот, типы работ, запуск, ограничения, отчёты, архитектура, БД, API, параметры облака, задачи планировщика |
