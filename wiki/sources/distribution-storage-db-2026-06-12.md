## Сущности БД

| Сущность | Описание |
|---------|---------|
| **Продукт** | Дистрибутив: объединяет версии одного приложения |
| **ВерсияПродукта** | Конкретная сборка дистрибутива |
| **УстановленнаяВерсия** | Версии, защищённые от удаления |
| **Сервис** | Соответствует ssrv, собранным под определённую архитектуру |
| **ПропатченныеСервисы** | Сервисы с применёнными патчами |
| **ФайлВерсии** | Общие файлы дистрибутива, не относящиеся к сервисам |
| **Скрипт** | Архив со скриптами (привязан к ВерсияПродукта) |
| **Патч** | Архив с патчами (привязан к ВерсияПродукта) |
| **ФайлХранилища** | Ссылка на filestorage + метаданные (размер, название, mime, md5) |
| **Модуль** | Модуль дистрибутива (smod), тип: БЛ/интерфейс |
| **ВерсияМодуля** | Версия модуля с VersionHash — зависит от версии платформы, алгоритма сборки |
| **СвязьСервисаСМодулями** | M2M: сервис ↔ его модули с версиями |
| **ЧастьСервиса** | Устаревшая реализация сущности Модуль |
| **Образ** | Артефакты ks: Docker-образы и Helm-чарты (связан с сервисом и ВерсияПродукта) |
| **ФайлОбновления** | Логи и файлы, создаваемые в процессе обновлений |
| **СервисХоттабыча** | Сервисы «Управление версиями» — разграничение доступа к ФайлОбновления |
| **СкриншотДистрибутива** | Скриншоты мобильных дистрибутивов |
| **ДоступКлиентовКДистрибутивам** | Связь ClientId автономных облаков с saby-пакетами |
| **СоставSabyПакета** | Дистрибутивы, входящие в saby-пакет |

ФайлХранилища расширяется таблицами: ФайлВерсии, Скрипт, Патч, ФайлОбновления, Module, ServicePart.

## Типовые выборки

- **Версия дистрибутива**: доступ по PK в ВерсияПродукта
- **Список версий**: по FK «Продукт», сортировка по Дата/Название/Сборка
- **Дерево файлов версии**: объединение ФайлВерсии + Module + Скрипт + Патч по FK ВерсияПродукта
- **Загрузка файла**: проверка по md5 (`ФайлХранилища(md)`) → дедупликация
- **Удаление файла**: подсчёт ссылок на ФайлХранилища → удаление из filestorage при 0 ссылок
- **Поиск модуля**: по (Name, Type, Revision) — уникальный составной ключ в ModuleVersion
- **Поиск образа**: по полям Url и Host в таблице Image

## Ключевые индексы

```
Module(Name, Type, Revision)            -- уникальность содержимого модуля
ModuleVersion(VersionHash, Name, Type)  -- уникальный (unique)
ServiceToModule(ModuleVersion, Module, Service)
Service(DistrVersionId, SystemName, Arch, OS, OSVersion)
ВерсияПродукта(Продукт, Дата DESC, Название DESC, Сборка DESC)
ВерсияПродукта(Продукт, Название, Сборка, Тип)
УстановленнаяВерсия(Облако, Приложение, Статус, ВерсияПродукта, ...)
ФайлХранилища(md)                       -- дедупликация
Image(Url)
DistrClient(ClientId, Продукт)          -- права автономных облаков
ServicePart(Revision), ServicePart(md)
```
