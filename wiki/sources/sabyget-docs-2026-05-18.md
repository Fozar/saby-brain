## Ключевые находки

- SabyGet — потребительская витрина системы лояльности Tensor: бонусные карты, промокоды, реферальная программа выходят именно через этот продукт
- Proxy-архитектура: все данные получаются proxy-вызовами через аккаунт `online.sbis.ru`, нет собственной БД
- Сортировка каруселей заведений имеет нетривиальный алгоритм весов (Избранное×1 + Клиент×10 + Затраты×100)
- Реферальная программа видна пользователям через диалог детализации бонусов
