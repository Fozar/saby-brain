# Source: Платформенный механизм кэширования отчетов (report-prefetch-service)

**Ingested:** 2026-04-13
**Domain:** Wasaby Backend Platform

## Summary

Официальная документация платформенного сервиса `report-prefetch-service` — механизма предварительного кэширования страниц больших отчётов в SBIS/Saby. Описывает API (`Prefetch.List`, `Prefetch.Data`, `Prefetch.AppendBatch`, `Prefetch.CreateSession`, `Prefetch.DiscardSession`), параметры фильтрации, работу с иерархией, сортировку с индексами, поиск, выбор записей и суммирование.
