# Saby API Docs — Auth & Tasks (2026-05-21)

Шесть страниц официальной документации Saby (saby.ru/help) по внешнему JSON-RPC API.

## Охваченные темы

| Источник | Содержание |
|---|---|
| Пройти аутентификацию с помощью API | Обзор: два метода входа, сессионный заголовок, ограничения |
| СБИС.Аутентифицировать | Метод login/password: параметры, ответы, коды ошибок, SMS 2FA |
| СБИС.Выход | Завершение сессии; 1-минутное окно после выхода |
| СБИС.ИнформацияОТекущемПользователе | Текущий пользователь + внешние провайдеры (Yandex, Esia) |
| СБИС.СписокЗадач | Список документов сотрудника из «Задачи/На мне» и «От меня» |
| Справочник кодов ошибок | classid-таблица: `1FA...` (fatal), `1AA...` (non-fatal) |

## Ключевые инсайты

- Сессионная аутентификация: ID сессии через `X-SBISSessionID`, одноразовое получение на весь сеанс.
- Rate limit: 300 вызовов/мин для auth методов → блокировка IP на 20 мин; 429 + classid `1FA000001000`.
- SMS 2FA (classid `1FA000001002`): флоу подтверждения через `СБИС.ОтправитьКодАутентификации` + `СБИС.ПодтвердитьВход`.
- `СБИС.СписокЗадач` — курсорная пагинация по `ДатаВремя` (unix timestamp с микросекундами).
- Нефатальные ошибки `1AA...` безопасно повторять; `1AA0000F1002` — ждать 3–5 сек (async result).
