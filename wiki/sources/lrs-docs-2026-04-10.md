# LRS Long Request Service Docs

4 документа о сервисе длительных операций (Long Request Service, LRS):

1. **Длительные операции** — обзор: LRS как надстройка над DWC, возможности, права.
2. **Поиск в логах** — работа с UUID-цепочками; специфика LRS (UUID не сохраняется, поиск через log_id:).
3. **Работа с длительными операциями на бизнес-логике** — создание в Genie, флаги, SetCustomData, формат результата, LongRequestNotify, прерывание, локализация. Примеры Python и C++.
4. **Техническая документация** — цели, архитектура, бесшовное обновление (3 режима + redis), шардирование БД (64 виртуальных шарда), схема таблиц, 10 типовых выборок, индексы, публичное API.

**Key insight**: LRS — тонкий слой поверх DWC, добавляющий UX (прогресс, результат, история) + персистентность (90 дней) + шардированную БД + права. DWC остаётся главным исполнителем.
